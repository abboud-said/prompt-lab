# 🚀 AI Prompt Lab (Gemini 1.5)

A professional AI prototyping dashboard built with React 19, Vite 8, and Tailwind CSS v4. This lab allows users to experiment with AI "Temperatures" and track their prompt history.

## ✨ Key Features
- **AI Integration**: Connected to Google Gemini 1.5 Flash for real-time responses.
- **State Management**: Uses `useReducer` for complex state transitions (Loading, Success, History).
- **Persistence**: `useEffect` hooks synchronized with `localStorage` to keep history after refresh.
- **Modern UI**: Fully responsive dark-mode design using Tailwind v4 and Lucide icons.

## 🛠️ Tech Stack
- **React 19** (Functional Components & Hooks)
- **Vite 8** (Lightning fast build tool)
- **Tailwind CSS v4** (Utility-first styling)
- **Google Generative AI SDK**

## 🚀 How to Run Locally
1. Clone the repo.
2. Run `npm install`.
3. Create a `.env` file and add `VITE_GEMINI_API_KEY=YOUR_KEY`.
4. Run `npm run dev`.# prompt-lab
